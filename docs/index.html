The Ultimate MIDI Bridge for Yamaha Synthesizers

Seamlessly connect your modern Windows PC to legacy Yamaha synthesizers via TO HOST serial ports.

ToHost Bridge provides high-performance, 80-channel MIDI multiplexing completely free and open source.

Features

🎹

5 Virtual Outputs

Multiplex up to 5 virtual MIDI out ports (80 MIDI channels) into a single high-speed serial stream for multi-part setups.

⚡

Zero Lag

Highly optimized C++ engine with real-time thread priority ensures rock-solid timing and low latency.

🛠️

MIDI 2.0 Ready

Built on the official Microsoft Windows MIDI Services SDK for the best compatibility and performance.

📦

Small & Portable

Runs as a single executable or installs via a per-user MSI without admin rights. Stealthy system tray mode included.

Technical Advantage

💾

Bulletproof SysEx

Verified stable with 256-byte bulk dumps that often crash budget USB-MIDI cables. Perfect for Yamaha Data Filer and bulk editing.

🚀

23% More Bandwidth

TO HOST operates at 38,400 baud.

This provides a 23% speed increase over the 31,250 baud of standard MIDI.

🔌

The Single-Cable Solution

Replace expensive multi-port interfaces and complex cabling with one high-speed Serial-to-USB link for all 64 channels.

⚠️

Standalone App Required

Unlike USB-MIDI interfaces with system-level drivers, you need the app to be running at all times to bridge MIDI data to your hardware.
