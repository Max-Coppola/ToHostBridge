The Professional MIDI to Yamaha Serial Bridge.

ToHost Bridge is free software. Connect your modern Windows PC to legacy Yamaha synthesizers via TO HOST serial ports. High performance, 4-port multiplexing (64 MIDI channels), and full MIDI 2.0 support.

Download Installer (.MSI) Download Standalone (.EXE)
Version 1.1 | Windows 10/11 x64
🎹

4 Virtual Outputs

Multiplex up to 4 virtual MIDI out ports (64 MIDI channels) into a single high-speed serial stream for massive multi-part MU-series setups.

⚡

Zero Lag

Highly optimized C++ engine with real-time thread priority ensures rock-solid timing and low latency.

🛠️

MIDI 2.0 Ready

Built on the official Microsoft Windows MIDI Services SDK for the best compatibility and performance.

📦

Small & Portable

Runs as a single executable or installs via a per-user MSI without admin rights. Stealthy system tray mode included.
